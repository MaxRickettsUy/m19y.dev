How to Configure Flutter and Android Studio on Linux Mint

4/15/2022

  1. Download Flutter (I used snap)
  2. Run flutter doctor
  3. Download Android Studio. Save to ~/Download
  4. `cd ~/Download` then `tar -xzf <android_studio_tgz>
  5. `sudo mv <android_studio_dir> /usr/local`
  6. Create symlink in /usr/local/bin : ln -s /usr/local//bin/studio.sh studio.sh. You will now be able to start from command line or i3bar with studio.sh
  7. Install Chrome
  8. Configure Android SDK path for Flutter: flutter config --android-sdk /home/<user_name>/Android/Sdk
  9. Use Android studio to get the remaining command line tools: StackOverflow instructions
  10. Set licenses: flutter doctor --android-licenses
  11. Run flutter doctor
