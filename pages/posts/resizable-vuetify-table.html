How to make a resizable Vuetify data table.

8/9/2021

I was recently tasked with using the Vuetify data table component. The component accepts a height prop (no percentage or vh). I was able to leverage this Stack Overflow question to get the job temporarily before I developed a better method while attempting to write unit tests for my custom Resizable component.

Rather than creating and managing a separate component to handle resizing with a slot, I realized I could use the tables container (a card in this case) and it's offsetHeight to update the data table's height prop during resize event. I am using a magic number to deduct from the offsetHeight to better fit the data table in the card.

With this dynamic height, the data table scales with the card container which has a height of 60vh. This way when the window is shortened, the data table remains within it's container and does not spill out of the container and potentially out of view.
